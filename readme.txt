WordPress Plugin to embedd klickTel Open API in your Wordpress blog 

== Description ==

= This plugin is an early beta version. Please don't use it on production systems (yet)! =

This plugin will give your blog access to the klickTel Open-API. 
klickTel is a brand of telegate MEDIA in germany, which provides free and paid directory listing to small and medium companies in germany. 
You can look up several kinds of information e.g. phone book entries and trade name based searches for businesses.

The intention of the plugin is to extend the existing search capabilities of WordPress to include the klickTel Open-API. 
That way you can look up directory listings for a specific search word or get the nearest restaurant to order some food.

== Installation == 
You can unzip the archive to your plugins folder, or use the wordpress installer.

= Requirements =
* 'allow_url_fopen' in php.ini must be set to 'On'

== Frequently Asked Questions ==

= Where do I get an Open-API key? =

You can register for the klickTel Open-API here: http://openapi.klicktel.de
